### ü™ê About The Project

I built this site to recapture the cozy nostalgia of the old web. It avoids modern minimalism in favor of **personality, texture, and fun**.

**‚ú® Key Visual Features:**

- **CSS Animations:** Floating stickers (`.floater`), blinkies, buttons and seamless scrolling marquees.
- **Responsive Design:** A locked "Window" view on Desktop, but a fully scrolling layout on Mobile.
- **Interactive Elements:** Hover glows, blinkie boards, and interactive pages with fun elements. (In Progress)
